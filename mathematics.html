Monte Carlo \(\pi\) Approximation

This demonstration shows how we can approximate \(\pi\) by repeatedly
throwing darts at a target (the black circle) inscribed in a square.
The idea is that the number of darts that actually hit the target
represents the area of a circle, \(A_{c}\), with radius \(r\) and
the total number of darts represents the area of a square, \(A_{s}\),
with length and width \(2r\). Using these approximations, we can get
a pretty good estimation of \(\pi\) in the following way: $$A_{c} = {\pi}r^{2}$$ $$A_{s} = (2r)^2 = 4r^2$$ $$\implies \frac{A_{c}}{A_{s}} = \frac{\pi}{4}$$ $$\therefore 4\frac{A_{c}}{A_{s}} = \pi$$

Showing that $$n^2 = \binom{n}{2} + \binom{n + 1}{2}$$

Number to be squared:



Although this is a nice geometric way to observe this property, it does
not suffice as proof. However, the proof is quite simple because it is
really just a bit of symbolic manipulation. Observe:
$$\binom{n}{2} + \binom{n + 1}{2} = \frac{n!}{(2!)(n - 2)!} + \frac{(n + 1)!}{(2!)(n - 1)!} \\ = \left(\frac{1}{2}\right)\left(\frac{n!}{(n - 2)!} + \frac{(n + 1)!}{(n - 1)!}\right) \\ = \left(\frac{1}{2}\right)((n)(n - 1) + (n + 1)(n)) \\ = \left(\frac{1}{2}\right)(n^2 - n + n^2 + n) \\ = \left(\frac{1}{2}\right)(2n^2) \\ = n^2$$
